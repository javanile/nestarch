## `nestarch help [COMMAND]`

Display help for nestarch.

```
USAGE
  $ nestarch help [COMMAND...] [-n]

ARGUMENTS
  COMMAND...  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for nestarch.
```
